🚀 Local Server cho Quiz Application

📋 Hướng dẫn Khởi động Server

🔧 Cách 1: Sử dụng Python (Khuyến nghị)

Mở Command Prompt hoặc PowerShell trong thư mục qa-viewer và chạy:

# Python 3 python -m http.server 8000 # Hoặc Python 2 python -m SimpleHTTPServer 8000

Sau đó mở trình duyệt và truy cập: http://localhost:8000

🔧 Cách 2: Sử dụng Node.js

Nếu bạn có Node.js, cài đặt http-server:

npm install -g http-server http-server -p 8000

Sau đó mở trình duyệt và truy cập: http://localhost:8000

🔧 Cách 3: Sử dụng Live Server (VS Code)

Nếu bạn sử dụng VS Code:

1. Cài đặt extension "Live Server" 2. Right-click vào index.html 3. Chọn "Open with Live Server"

❓ Troubleshooting

Lỗi thường gặp:

  • Port 8000 đã được sử dụng: Thay đổi port khác (8001, 8080, etc.)
  • Python không được cài đặt: Tải và cài đặt Python từ python.org
  • Firewall chặn: Cho phép Python/Node.js qua firewall
  • File không tồn tại: Đảm bảo file questions.json có trong thư mục
